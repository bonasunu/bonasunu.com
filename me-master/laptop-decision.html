Laptop Decision


Open on Github

Background Story

This web app inspiration came from cameradecision.com. Then I find that it would be very useful for many people if they have resource to search informations about laptops. For this reason, I build this web app.


Tech Stacks

I use Django and SQL for this web app. The main reason I choose the stacks is the design from Django that help me take application from concept to completion as quickly as possible (Minimum Viable Product).


Challenges

There are 2 challenges to build this app:

  • - Algorithms for laptop recommendation on help-me-choose page.

  • - Smart filter for relational database


Implementation

Here are my implementation for this web app to build main features:

  • - Build simple form for user preference to help user find laptop. Then I use the user preferences to filter laptops.

  • - Smart filter using django filter module
